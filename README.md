# 🚀 Gumroad File Limit Extender (20GB → 2TB)

**Unlock the full potential of your Gumroad store** by bypassing the standard 20GB upload restriction. This specialized application allows creators to host and sell massive datasets, high-definition video courses, and ultra-high-resolution assets up to **2TB per file.**

## 🌟 Key Features

- **Massive Storage Increase:** Effortlessly move beyond the **20GB limit** all the way up to **2,000GB (2TB)**.
- **Seamless Integration:** Designed to work with your existing Gumroad workflow.
- **High-Speed Uploads:** Optimized protocol for handling **large-scale data transfers** without timeouts.
- **Creator Freedom:** Ideal for game developers, filmmakers, and VFX artists who deal with **massive file sizes**.

## 🛠 How It Works

This tool utilizes a custom API bridge that modifies the upload handshake, allowing the Gumroad backend to accept **extended file headers**. Once implemented, your dashboard will reflect the **newly expanded capacity**.
